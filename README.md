# 纳斯达克100指数监控系统

## 项目简介
这是一个实时监控纳斯达克100指数的Web应用，支持多种时间范围的回撤分析和数据可视化。

## 核心功能
- 📈 **实时数据**：获取纳斯达克100指数真实数据
- 📊 **多时间范围**：支持1年、3年、5年、10年、15年、20年
- 📉 **回撤分析**：计算当前回撤和最大回撤
- 🏆 **历史最高点**：标记历史最高指数值和日期
- 📱 **响应式设计**：支持PC和移动设备
- 🖱️ **交互功能**：鼠标/触摸显示指数值
- 🔄 **自动更新**：每24小时自动更新数据

## 技术栈
- **前端**：HTML5, CSS3, JavaScript, SVG图表
- **后端**：Node.js, Express
- **数据来源**：Eastmoney API, Yahoo Finance API, Stooq API
- **部署**：本地部署，支持内网穿透

## 快速开始

### 环境要求
- Node.js 10.0+
- npm 6.0+

### 安装依赖
```bash
npm install
```

### 启动服务
```bash
node server.js
```

### 访问地址
- 本地访问：http://localhost:8080
- 内网访问：http://[本机IP]:8080

## 功能说明

### 时间范围选择
- 1年：显示最近1年的交易日数据
- 3年：显示最近3年的交易日数据
- 5年：显示最近5年的交易日数据
- 10年：显示最近10年的交易日数据
- 15年：显示最近15年的交易日数据
- 20年：显示最近20年的交易日数据

### 指标解释
- **当前回撤**：从最近一次历史最高点到当前的回撤
- **最大回撤**：选定时间范围内的最大回撤
- **历史最高**：选定时间范围内的最高指数值
- **形成天数**：回撤形成的天数
- **修复天数**：回撤修复的天数

### 图表说明
- **蓝色曲线**：纳斯达克100指数走势
- **橙色区域**：当前回撤区间
- **红色区域**：最大回撤区间
- **绿色区域**：最大回撤修复区间
- **橙色点**：当前回撤的起点（历史最高点）
- **绿色点**：最大回撤的起点（最大值点）
- **红色点**：最大回撤的终点（最小值点）
- **绿色点**：最大回撤的修复点（回到最大值点）

## 数据获取
- 优先使用 Eastmoney API 获取纳斯达克100指数数据
- 备用使用 Yahoo Finance API
- 最终备用使用 Stooq API
- 当所有API都失败时，使用模拟数据
