# 智能创作

## 项目简介
这是一个实时监控美股指数的Web应用，支持纳斯达克100和标普500指数的回撤分析和数据可视化。

## 核心功能
- 📈 **实时数据**：获取纳斯达克100和标普500指数真实数据
- 📊 **多时间范围**：支持1年、3年、5年、10年、15年、20年
- 📉 **回撤分析**：计算当前回撤和最大回撤
- 🏆 **历史最高点**：标记历史最高指数值和日期
- 📱 **响应式设计**：支持PC和移动设备
- 🖱️ **交互功能**：鼠标/触摸显示指数值
- 🔄 **自动更新**：每24小时自动更新数据
- 📊 **指数对比**：支持纳斯达克100和标普500对比分析

## 技术栈
- **前端**：HTML5, CSS3, JavaScript, SVG图表
- **后端**：Node.js, Express, SQLite
- **数据来源**：Stooq API
- **部署**：本地部署，支持内网穿透

## 功能说明

### 指数选择
- **纳斯达克100**：NDX指数，科技股权重较高
- **标普500**：SPX指数，美股大盘指数
- **对比模式**：同时显示两个指数的走势对比

### 时间范围选择
- 1年：显示最近1年的交易日数据
- 3年：显示最近3年的交易日数据
- 5年：显示最近5年的交易日数据
- 10年：显示最近10年的交易日数据
- 15年：显示最近15年的交易日数据
- 20年：显示最近20年的交易日数据

### 指标解释
- **当前回撤**：从最近一次历史最高点到当前的回撤
  - 计算公式：`(历史最高 - 当前值) / 历史最高 × 100%`
  - 形成天数：从历史最高点到当前的天数
- **最大回撤**：选定时间范围内的最大回撤
  - 计算公式：`(最大值 - 最小值) / 最大值 × 100%`
  - 形成天数：从最大值点到最小值点的天数
  - 修复天数：从最小值点回到最大值点的天数
- **历史最高**：选定时间范围内的最高指数值及日期

### 图表说明
- **蓝色曲线**：纳斯达克100指数走势
- **橙色曲线**：标普500指数走势（对比模式）
- **橙色区域**：当前回撤区间
- **红色区域**：最大回撤区间
- **绿色区域**：最大回撤修复区间
- **橙色点**：当前回撤的起点（历史最高点）
- **绿色点**：最大回撤的起点（最大值点）
- **红色点**：最大回撤的终点（最小值点）

## 数据获取
- 使用 Stooq API 获取纳斯达克100和标普500指数数据
- 数据更新频率：每24小时自动更新
- 数据存储：本地SQLite数据库缓存
