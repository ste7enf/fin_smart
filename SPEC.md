# 智能创作 - 指数监控系统规格说明

## 1. 项目概述

### 项目名称
智能创作 - 美股/ETF指数监控系统

### 项目类型
Web应用 (Node.js + Express + SQLite)

### 核心功能
实时监控纳斯达克100指数、标普500指数和红利低波ETF的回撤分析，支持多时间范围数据可视化和对比分析。

### 目标用户
投资者、量化交易者、金融分析师

## 3. 功能规格

### 3.1 核心功能

#### 指数数据获取
- **数据源**:
  - 纳斯达克100 (^ndx): Stooq API
  - 标普500 (^spx): Stooq API
  - 红利低波ETF (512890): 东方财富API
- **更新频率**: 每24小时自动更新
- **手动刷新**: 支持点击刷新按钮触发更新

#### 回撤分析
- **当前回撤计算**:
  - 公式: (历史最高 - 当前值) / 历史最高 × 100%
  - 记录历史最高点日期
  - 计算形成天数（从最高点到当前的天数）
- **最大回撤计算**:
  - 遍历整个数据范围找到最大跌幅
  - 记录最大值日期、最小值日期
  - 计算形成天数和恢复天数

#### 时间范围选择
- 支持: 1年、3年、5年、10年、15年、20年
- 默认: 1年
- 切换时自动重新计算指标

#### 对比模式
- 同时显示纳斯达克100和标普500的归一化曲线
- 以第一个数据点为基准 (=100)
- 展示两个指数的相对表现

### 3.2 用户交互

#### 指数切换
- 点击指数按钮切换显示
- 切换时显示加载状态
- 图表平滑过渡动画

#### 时间范围切换
- 点击时间按钮切换
- 自动重新计算所有指标

#### 图表交互
- 鼠标悬停显示详细数据（日期、价格）
- 触摸支持移动端交互

### 3.3 API 接口

#### GET /api/indexes
- **参数**:
  - `symbol`: ndx | spx | h30269 | compare (默认 ndx)
  - `years`: 1 | 3 | 5 | 10 | 15 | 20 (默认 1)
- **响应**:
  ```json
  {
    "index": {
      "name": "纳斯达克100指数",
      "symbol": "ndx",
      "data": [price1, price2, ...],
      "dates": ["2025-01-01", "2025-01-02", ...],
      "current_drawdown": 0.1234,
      "peak": 20000,
      "peak_date": "2025-12-15",
      "formation_days": 50,
      "max_drawdown": 0.2345,
      "max_value": 21000,
      "min_value": 16000,
      "max_date": "2025-02-01",
      "min_date": "2025-04-01",
      "recovery_days": 30
    },
    "last_update": "2026-03-04 12:00:00"
  }
  ```

#### GET /api/refresh
- 触发手动数据刷新
- 返回: `{ status: 'ok', message: '数据已刷新' }`

## 4. 数据流

### 4.1 启动流程
1. 服务器启动
2. 初始化SQLite数据库
3. 串行获取各指数数据:
   - 先获取纳斯达克100 (延迟2秒)
   - 再获取红利低波 (延迟2秒)
   - 最后获取标普500
4. 数据加载完成，设置 isDataReady = true
5. 响应API请求

### 4.2 数据获取失败处理
- 不使用模拟数据
- 设置 dataLoadError 错误信息
- API返回503状态码和错误消息
